# Nomadikas — Homepage Preview

Design prototype for the new Nomadikas homepage. Self-contained single-file HTML.

Live preview: [vincentbraakman-glitch.github.io/nomadikas-homepage-preview](https://vincentbraakman-glitch.github.io/nomadikas-homepage-preview/)

## What this is

A full-page design prototype of the Nomadikas homepage, built as a single self-contained HTML file with inline CSS and JS. Hot-links imagery from public sources. Opens offline via `file://`.

Once the design is approved, the page gets translated into an Elementor template for the production WordPress site.

## Editing

Open `index.html` in any editor. Commit + push — GitHub Pages rebuilds within a minute.
